## ⚙️ Quickstart

1. Install `gtkmm4` library by executing `brew install gtkmm4`
2. Build the Make project using the `make` command
3. Execute the binary with `./project`
4. Select a test file under the `tests` directory
5. Start the simulation by pressing "start"
